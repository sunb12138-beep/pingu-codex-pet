# Pingu 桌宠

一个基于经典瑞士定格黏土动画版 Pingu 形象制作的 Codex 自定义桌宠。

它保留了原作里那种圆润、简洁、很有表情的企鹅气质，同时把动作做成了适合桌面陪伴的动画状态：会待机、会走动、会挥手、会跳跃，也会在不同方向上自然地转头观察。

## 特点

- 还原经典 Pingu 的黑白配色、红色喙和橙色脚掌
- 采用更接近定格黏土感的外形，而不是 3D 风格
- 支持完整的 11 行动画表情/动作图集
- 16 个方向看向动画，适合桌面交互场景
- 动作强调物理规律，避免夸张扭曲和不自然变形

## 动作说明

- `idle`：安静待机，轻微呼吸/眨眼
- `running-right` / `running-left`：左右移动
- `waving`：挥手招呼
- `jumping`：跳跃
- `failed`：失败或沮丧反馈
- `waiting`：等待你的下一步指令
- `running`：正在处理任务
- `review`：检查和观察结果
- 方向看向：用于表现头部和视线转向

## 使用场景

- 作为 Codex 的自定义桌宠
- 作为 GitHub 项目展示的个性化角色
- 作为桌面陪伴型动画资产
- 作为一套可继续扩展的角色动作基础

## 说明

这是一个以经典 Pingu 为灵感的桌宠项目，重点是“像原作”，而不是做成夸张卡通化版本。  
如果你喜欢那种安静但又很有生命力的企鹅桌宠，这个版本会比较适合。
